### 🐝 Life of Bees – Base de Datos para Investigación Apícola

**Proyecto final para el certificado CS50: Introduction to Databases with SQL de HarvardX.**

- **Objetivo:** Diseñar, implementar y documentar una base de datos relacional con SQLite para gestionar información sobre la vida y comportamiento de las abejas.
- **Proceso:** Seguí el ciclo completo de desarrollo de una base de datos: análisis de requisitos, diseño de esquema (diagrama entidad-relación), implementación de tablas y consultas SQL.
- **Resultado:** Obtuve el certificado oficial de CS50.
- **Video de presentación del proyecto:** [Life of Bees - CS50 Project](https://youtu.be/1FFk6nv4IQQ?si=RfwQfUtfrMXQGSMQ)
- **Tecnologías:** SQLite, SQL
